Features

Link Fingerprint

  • Link fingerprint to existing prepaid number.
  • Link fingerprint to existing postpaid number.
  • Link fingerprint to existing Data SIM.

Transfer Ownership

  • Transfer prepaid (Sawa + Data SIM) ownership to existing STC customer.
  • Transfer prepaid (Sawa + Data SIM) ownership to new customer.

New SIM Card

  • Issues new Sawa SIM card.
  • Issues new prepaid Data SIM card.
  • Issues new Sawa Zeyara SIM card.
